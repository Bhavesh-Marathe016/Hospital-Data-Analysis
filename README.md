# 🏥 Hospital Data Analysis Dashboard | Power BI

## 📌 Project Overview

The **Hospital Data Analysis Dashboard** is an interactive Business Intelligence solution developed using **Power BI** to analyze hospital operations and healthcare performance. The dashboard transforms raw hospital data into meaningful insights, enabling administrators and healthcare professionals to monitor patient trends, improve operational efficiency, and support data-driven decision-making.

---

## 🎯 Project Objectives

* Analyze patient admissions and discharge trends.
* Monitor hospital occupancy and resource utilization.
* Track doctor and department performance.
* Identify patient demographics and treatment patterns.
* Measure revenue and billing performance.
* Provide interactive reports for better healthcare decision-making.

---

## 🛠️ Tools & Technologies

* **Power BI** – Dashboard Development & Visualization
* **Power Query** – Data Cleaning & Transformation
* **DAX** – KPI Measures & Calculations
* **Data Modeling** – Table Relationships
* **Microsoft Excel/CSV** – Data Source

## 🔍 Business Questions Solved

* How many patients were admitted during the selected period?
* Which department treated the most patients?
* Which doctors handled the highest number of patients?
* What is the average hospital stay?
* Which age group has the highest patient count?
* What is the hospital's total revenue?
* What are the monthly admission and discharge trends?
* Which departments have the highest bed occupancy?

## 💡 Key Insights

* Identified departments with the highest patient load.

* Measured hospital revenue and treatment costs.
* Analyzed bed occupancy to improve resource planning.
* Evaluated doctor performance based on patient volume.
* Improved visibility into hospital operations through interactive dashboards.

---

## 🔄 Project Workflow

1. Imported hospital dataset into Power BI.
2. Cleaned and transformed data using Power Query.
3. Built relationships between tables using Data Modeling.
4. Created DAX measures for KPIs and calculations.
5. Designed interactive dashboards with charts, cards, and slicers.
6. Tested dashboard accuracy and optimized performance.
7. Published the final Power BI report.
